# Sport News Website

A modern, clean sports news website built with React and Tailwind CSS.

## Features

- Responsive design matching the original layout
- Clean header with navigation and search functionality
- Main feature section with large hero image
- Sidebar with today's news articles
- Modern grayscale design aesthetic

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The application will be available at `http://localhost:5173`

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Technologies Used

- React 18
- Tailwind CSS 3
- Vite
- Google Fonts (Dancing Script for logo)

## Project Structure

```
src/
  components/
    Header.jsx      # Navigation header component
    MainFeature.jsx # Main hero section component
    Sidebar.jsx     # Today's news sidebar component
  App.jsx           # Main app component
  main.jsx          # Entry point
  index.css         # Tailwind CSS imports
```
